What is an SPA?

A Single Page Application loads a single HTML document and updates the UI as the user navigates, without full page refreshes. Routing is handled in JavaScript to swap views based on the URL.

React Router: Key Pieces

Components

  • <BrowserRouter>
  • <Routes> & <Route>
  • <Link> / <NavLink>
  • <Outlet> (nested routes)

Hooks

  • useNavigate()
  • useParams()
  • useLocation()
  • useSearchParams()

Minimal Example

import { BrowserRouter, Routes, Route, Link } from "react-router-dom";

function Home() { return <h2>Home</h2>; }
function About() { return <h2>About</h2>; }

export default function App() {
  return (
    <BrowserRouter>
      <nav>
        <Link to="/">Home</Link> | <Link to="/about">About</Link>
      </nav>

      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}
