Intro to the useParams Hook

Lesson

Learning Objectives

Students will be able to:

  • Apply route params to create a dynamic route and navigate to multiple template views on a site

What is React Router DOM?

React Router DOM is a library used to handle routing in React applications. It allows developers to create Single Page Applications (SPAs) with navigation that doesn't require a page reload.

Why Use the useParams Hook?

The useParams hook is a part of React Router DOM that allows you to access the parameters of the current route. It is particularly useful for extracting dynamic parameters from the URL, such as IDs, which can then be used to fetch data or display specific content based on the URL.

Accessing Route Parameters with useParams

Let's explore how to use the useParams hook to access route parameters through a practical example.

Basic Setup

We'll start by setting up our router with some basic routes. Here is an example of an App component with routing and navigation. 


import './App.css';
import Home from './components/Home';
import Navbar from './components/Navbar';
import InventoryPage from './components/InventoryPage';
import InventoryDetails from './components/InventoryDetails';
import { Routes, Route } from 'react-router-dom';

function App() {
  return (
    <>
      <Navbar />
      <Routes>
        <Route path='/' element={<Home />} />
        <Route path='/inventory' element={<InventoryPage />} />
        <Route path='/inventory/:id' element={<InventoryDetails />} />
      </Routes>
    </>
  );
}

export default App;

In this setup, we have defined three routes:

  • / for the Home page.
  • /inventory for the Inventory page.
  • /inventory/:id for the Inventory Details page, where :id is a dynamic parameter.

Accessing Route Parameters with useParams

Next, we'll use the useParams hook in the InventoryDetails component to access the dynamic id parameter from the URL. 


import { useParams } from 'react-router-dom';

function InventoryDetails() {
  let { id } = useParams();

  return (
    <div>
      <h2>Inventory Details</h2>
      <p>Item ID: {id}</p>
      {/* Use the id to fetch or display more details about the inventory item */}
    </div>
  );
}

export default InventoryDetails;

Explanation

  • The useParams Hook: The useParams hook is used to access the dynamic parameters of the current route. In this case, let { id } = useParams(); extracts the id parameter from the URL.
  • Using Route Parameters: The extracted id parameter can be used to fetch or display specific details about an item. For example, in the InventoryDetails component, we display the id parameter directly in the UI. In a real application, you might use this id to fetch data from an API or a database.

Conclusion

The useParams hook in React Router DOM is a powerful tool for accessing dynamic route parameters in React applications. It allows developers to extract and use parameters from the URL, enabling the creation of more dynamic and responsive applications. By using useParams, you can easily fetch and display data based on the current route, enhancing the functionality of your SPAs.
