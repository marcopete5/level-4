Using useNavigate in React Router DOM

Lesson

Learning Objectives

Students will be able to:

  • Implement the useNavigate hook and assess its use cases

In this lesson, we will explore the useNavigate hook provided by React Router DOM. This hook is essential for programmatically navigating between different routes in a React application. We will learn how to use useNavigate to navigate to a specific route and how to move backward or forward in the browser history.

Basic Usage of useNavigate

The useNavigate hook returns a function that can be used to navigate programmatically. This function allows you to change the route based on certain actions or logic in your application. For instance, you can navigate to a specific route when a user clicks a button, submits a form, or after a successful API call.

  • Example: You might use useNavigate to navigate to an inventory item page when a user clicks on an item in a list. By calling the navigate function with the appropriate route, you can dynamically change the view based on user interaction.

Navigating Backward or Forward in History

You can also use useNavigate to move backward or forward in the browser's history. This functionality is helpful for providing a more intuitive navigation experience, such as allowing users to go back to the previous page or move forward to a page they visited before.

Example Code:


const navigate = useNavigate();

function goBack() {
    navigate(-1);  // Navigate back one page
}

function goForward() {
    navigate(1);  // Navigate forward one page
}

Explanation:

  • Navigating Back:
    • The navigate(-1) function call navigates one page back in the browser history.
    • This is equivalent to the user clicking the browser's back button.
  • Navigating Forward:
    • The navigate(1) function call navigates one page forward in the browser history.
    • This is equivalent to the user clicking the browser's forward button.

Usage:

Attach these functions to events to control navigation.


<button onClick={goBack}>Go Back</button>
<button onClick={goForward}>Go Forward</button>

Putting It All Together

  1. Navigate to a Specific Route:
    • Use the navigate function to go to a specific route based on some logic or user action.
  2. Navigate Backward or Forward:
    • Use the navigate function with 1 or -1 to control the browser history.

Conclusion

In this lesson, we learned how to use the useNavigate hook from React Router DOM to programmatically navigate between routes. We explored how to navigate to a specific route and how to move backward or forward in the browser history. These techniques are fundamental for building a dynamic and responsive navigation system in your React application.
