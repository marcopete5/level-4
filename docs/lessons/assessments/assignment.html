Level 4 Assignment Review

Assignment Review

You are to schedule a level assignment review after completion of the Changing Themes Using Context Assignment in level 4.

Assignments

  • Ugly Things
  • Changing Themes using Context
  • Roto Router

Requirements

  • All three assignments must be completed and ready to show to the best of their ability.
  • Students should be prepared to talk about the concepts and functionality used in each assignment.
  • Assignments will be reviewed and feedback will be sent back to the student.

Review Process

  • The review will be conducted in a one-on-one setting with an instructor during your End of Level Assessment.
  • The instructor will ask questions about the code and the student's understanding of the concepts.
  • The instructor will also give feedback on the student's work.

Benefits of the Review

  • The review is an important part of the learning process.
  • It's a chance for students to get feedback on their work and to improve their skills.
  • The review also helps students to prepare for grading of future assignments and assessments.

Questions?

Please contact the instructor if you have any questions about the Level Assignment Review.

Additional Notes

  • The instructor may ask students to share their screen during the review.
  • The call will be recorded as part of the learning process.

Submitting Assignments

Please remember to regularly submit your assignments to GitHub!

Using Git Command Line:

  1. Open Terminal and navigate to your project folder.
  2. Add Changes: git add <relative/path/to/assignment/folder>
  3. Commit Changes: git commit -m "submitting assignment <name>"
  4. Push to GitHub: git push origin main

Using VSCode's Source Control:

  1. Open VSCode and click on the Source Control tab.
  2. Stage Changes: Click the "+" icon next to each file.
  3. Commit Changes: Enter a commit message and click Commit.
  4. Push to GitHub: Click the Sync Changes button.
